# Rain Animation Website

A visually captivating rain animation website with ambient rain sound, lightning effects, and interactive user controls. Built with HTML, CSS (TailwindCSS), and JavaScript.

## Features

- **Rain Animation**: Realistic raindrops falling across the screen.
- **Lightning Effects**: Random lightning flashes to add dynamic effects.
- **Ambient Rain Sound**: Soothing rain sound playing in the background.
- **User Controls**: 
  - Play/Pause button for the rain sound.
  - Toggle button to turn the rain animation on or off.
  - Volume control slider for adjusting the rain sound.
